# مُرْتَقَى | منصة تسميع ومراجعة القرآن الكريم

- **مُرْتقى** أداة تم صنعها عن طريق الذكاء الاصطناعي لمساعدة حفظة كتاب الله مراجعة وتعزيز حفظهم ولمساعدة كل من يريد حفظ كتاب الله.
- تم استعمال **Alquran.cloud API** لجلب الآيات والتلاوات الصوتية لضمان الدقة وتوفير خيارات متعددة للمرتلين.

1. تثبيت التبعيات:
  `npm install`
2. تشغيل التطبيث:
  `npm run dev`
